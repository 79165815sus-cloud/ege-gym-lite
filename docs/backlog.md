# Бэклог

## Эпик 1. Контент и база заданий

**Цель:** актуальная и полная база заданий, корректные тексты и картинки.

### Таски

1) **Импорт новых вариантов**
- DoD:
  - Данные импортируются в `tasks.json` без потерь.
  - ID уникальны, старые задания не затираются.
  - `subject`, `examType`, `variant`, `examTaskNumber` заполнены.

2) **Проверка полноты заданий по предметам**
- DoD:
  - Русский: 27 заданий.
  - Математика профиль: 19 заданий.
  - Биология: 28 заданий.
  - Химия: 34 задания.

3) **Очистка текста и форматирование**
- DoD:
  - Переносы строк читаемы, нумерация не слипается.
  - LaTeX‑формулы отображаются корректно.
  - Нет мусорных вставок/метаданных из PDF.

4) **Связка картинок с заданиями**
- DoD:
  - `imageUrl` и `solutionImageUrl` заполнены по всем нужным задачам.
  - Картинки открываются в модалке и видны на странице.

5) **Валидация базы**
- DoD:
  - Нет дубликатов ID.
  - Нет пустых обязательных полей.
  - Проверка проходит без ошибок.

## Эпик 2. Прохождение и логика тренировки (тренажер)

**Цель:** стабильное прохождение в обоих режимах, корректный прогресс и итог.

### Таски

1) **Последовательный режим (Пробники)**
- DoD:
  - «Следующая» всегда переходит к следующему заданию.
  - Прогресс и цепочка синхронизированы с текущим номером.
  - При достижении конца показываются итоги.

2) **Рандомный режим (Отдельные задания)**
- DoD:
  - При нажатии «Следующая» загружается новое задание.
  - Счетчик «Задание X из Y» корректен.
  - По достижении лимита показываются итоги.

3) **Подарок после 5‑го задания**
- DoD:
  - Появляется ровно один раз за сессию.
  - После «Продолжить» возвращает на следующее задание.

4) **Ручная проверка (сложные задания)**
- DoD:
  - Для математики 13–19 и химии 29+ нет автопроверки.
  - Можно открыть решение и сохранить балл.
  - Баллы учитываются в итоговом экране.

5) **Итоговый экран тренировки**
- DoD:
  - Сумма баллов считается корректно.
  - Таблица содержит все задания тренировки, включая 0 баллов.

6) **Навигация по цепочке**
- DoD:
  - Клик по номеру переносит на нужное задание.
  - Текущий номер подсвечивается, пройденные отмечены.
