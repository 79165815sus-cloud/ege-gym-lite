# Качалка — тренажер ЕГЭ

Небольшое веб‑приложение на Go для тренировки заданий ЕГЭ с фильтрами,
последовательным режимом и простой админкой.

## Быстрый старт

```bash
go run main.go
```

Сервер поднимется на `http://localhost:8080`.

## Основные страницы

- `http://localhost:8080/` — «Качалка» (основной режим).
- `http://localhost:8080/filters.html` — план на тренировку.
- `http://localhost:8080/admin.html` — админка задач.
- `http://localhost:8080/admin-analytics.html` — аналитика (админка).
- `http://localhost:8080/gift.html` — экран подарка.
- `http://localhost:8080/stats.html` — статистика.
- `http://localhost:8080/simple.html` — упрощенная версия.

## Данные и структура

- `tasks.json` — база заданий.
- `static/` — все страницы и статика.
- `static/uploads/` — загруженные картинки.
- `analytics.jsonl` — лог событий (локально, игнорируется в Git).

## Аналитика

Клиент отправляет события в `/api/track`, лог пишется в `analytics.jsonl`.
Админ‑страница читает их через `GET /api/analytics`.

## Картинки в заданиях

В админке можно загрузить картинку — она сохраняется в `static/uploads/`
и автоматически подставляется в задание.

## Примечания

- Если добавляете новые поля в задания — обновляйте `tasks.json`.
